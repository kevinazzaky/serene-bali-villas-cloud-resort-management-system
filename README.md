# Serene Bali Villas — Cloud Resort Management System

Cloud-based Resort & Villa Management System untuk Serene Bali Villas.

## Project Overview

Project ini bertujuan membangun sistem manajemen resort dan villa berbasis cloud yang mendukung:
- Online booking
- Real-time room availability
- User authentication & role management
- Admin dashboard
- Payment gateway Stripe & Midtrans
- Email/WhatsApp notification
- Multi-language dan multi-currency
- Cloud deployment menggunakan AWS

## Methodology

Project menggunakan Agile/Scrum dengan sprint 2 minggu.

### Scrum Activities

| Activity | Frequency |
|---|---|
| Sprint Planning | Awal setiap sprint |
| Daily Standup | Setiap hari, 15 menit |
| Sprint Review | Akhir sprint |
| Sprint Retrospective | Akhir sprint |
| Backlog Grooming | Minggu ke-2 setiap sprint |

## Tech Stack

- Backend: Laravel
- Frontend: React.js + Tailwind CSS
- Database: PostgreSQL
- Cloud: AWS EC2, RDS, S3
- Payment: Stripe, Midtrans
- CI/CD: GitHub Actions
- API Docs: Postman / Swagger
- Project Management: GitHub Projects

## Sprint 1 Goal

Membangun fondasi sistem cloud-based yang mencakup:
- Booking online
- User authentication
- AWS infrastructure
- Real-time reservation support
- Multi-currency and multi-language support

## Definition of Done

A task is considered done when:
- Code review passed
- Unit test passed
- QA approved
- Documentation updated
